## Bootstrap

- väldigt populärt
- vill slippa uppfinna hjulet
- bibliotek för komponenter
- löste responsivitet väldigt tidigt
- var svårt med layouts tidigare, för 10 år sedan hade man table för layouts
- idag bygger bootstrap på flexbox
- bra stöd för SCSS
